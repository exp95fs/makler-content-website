Labelled form fields on warm surface with a terracotta focus ring; use for contact and enquiry forms.

```jsx
<Input label="Name" placeholder="Ihr Name" />
<Input label="E-Mail" type="email" error="Bitte gültige Adresse" />
<Textarea label="Ihr Objekt" rows={4} hint="Kurz beschreiben" />
```

`Input` supports `hint` and `error` (error turns the border terracotta-deep). `Textarea` matches the same styling and is vertically resizable.
