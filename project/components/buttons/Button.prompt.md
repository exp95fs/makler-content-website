Primary action control — terracotta CTA, sage secondary, terracotta-outline ghost; use for any call-to-action or form submit.

```jsx
<Button variant="primary" size="md">Termin anfragen</Button>
<Button variant="secondary">Portfolio ansehen</Button>
<Button variant="ghost">Preise &amp; Pakete</Button>
```

Variants: `primary` (terracotta, the main CTA — sparing, the 10%), `secondary` (sage), `ghost` (terracotta outline), `link` (inline text action). Sizes `sm | md | lg`. Pass `iconLeft` / `iconRight` Lucide SVGs. Hover deepens the colour; disabled drops to 45% opacity.
